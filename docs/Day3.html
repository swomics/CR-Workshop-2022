Active learning time!

In the previous sessions, we’ve had some hands on time running different programs and hopefully gaining an idea of the types of information we can extract from genomic data. Today I would like you to split into groups and attempt to formulate an evolutionary hypothesis that could be investigated using genomic data and the methods we have covered.

You’ve learned how to use genomic data to estimate heterozygosity for an organism, how to look for endosymbiotic bacteria, how to find genes and organise them into orthologous groups and trees. You’ve also learned how to compare genomes at a chromosome level. Using the DToL Lepidoptera species, we can think about linking these types of data across multiple species to a database of ecological traits. This database includes data on long and short term population size trends, host-plant specificity, phenological shifts and more!

You don’t have to worry about completing the project rigorously, this is simply an opportunity to begin thinking about how genomic data can be applied to ask biological questions

Here are some example hypotheses you could think about:

  • Do long term population declines predict reduced levels of heterozygosity? (paper)

  • Do polyphagous species have greater numbers of duplicated genes? (paper)

  • Do species that pupate in shared environments share similar endosymbionts? (paper)

  • Do closely related species have similar genome size? (paper)

You will need to think about how you will gather the genomic and ecological data, and how your resulting graphs might look.
